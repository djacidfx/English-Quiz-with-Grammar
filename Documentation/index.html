Getting Started : Java & Android Studio Installation - top

Changing App Name & About Text - top

1. Open app/src/main/res/values/strings.xml .
2. Change the value of String app_name To your Own App name
3. Change the value of String about_text to your own app about.




Changing App Icon - top



1. Open app/src/main/res/mipmap/ic_launcher icon.
2. Make your own icons.
3. Replace the ic_launcher.png by your Own icon with the same size.

Icon is must to upload application on play store. Bellow is the list of icon name with it's size. It must be in PNG file format.
* Store Icon.png: 512 x 512 pixel (for PlayStore)
* Folder name:-> mipmap-hdpi and icon name: ic_launcher.png 72 x 72 pixel
* Folder name:-> mipmape-mdpi and icon name: ic_launcher.png 48 x 48 pixel
* Folder name:-> mipmap-xhdpi and icon name: ic_launcher.png 96 x 96 pixel
* Folder name:-> mipmap-xxhdpi and icon name: ic_launcher.png 144 x 144 pixel
* Folder name:-> mipmap-xxxhdpi and icon name: ic_launcher.png 192 x 192 pixel.

Refrence site:-https://developer.android.com/guide/practices/ui_guidelines/icon_design.html

Changing privacy policy - Store link - Email - top

1 - Go to app/src/main/assets/privacy/index.html
2 - Open it with Notepad++ and Replace "CompanyName" to your own company name

Don't forget to change privacy policy URL :

1. Go to app/src/main/res/values/strings.xml.
2. Enter your URL inside "privacy policy" string tag : www.site.com/privacy_policy.html
3. Change Store URL link to your Developer Store : https://play.google.com/store/apps/developer?id=xxxxxx
4. Change Email to your Developer Email to receive Feedback from users : company@example.com
